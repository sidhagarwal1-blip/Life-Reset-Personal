# Life Reset Personal Mobile

This is an Expo starter scaffold for a personal-use mobile version of the app.

## Included in the scaffold

- Dashboard with day, streak, XP, and chart
- Daily quests with checkoff flow
- Weekly progression view
- Journal screen
- Pomodoro tool
- Reminder settings structure
- Export preview structure

## Next steps to make it real

1. Install dependencies with `npm install` or `npx expo install`.
2. Add `@react-native-async-storage/async-storage` for persistence.
3. Wire `expo-notifications` for scheduled local reminders.
4. Add file export and import with Expo file APIs.
5. Split `App.tsx` into screens and components.

## Run

```powershell
npm install
npx expo start
```

I did not install packages in this workspace, so this folder is a scaffold ready for the next pass.
